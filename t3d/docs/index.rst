The T3D code: documentation home
================================

T3D (or Trinity3D) is a transport solver for modeling macro-scale profile evolution in fusion plasmas (tokamaks and stellarators).

The T3D repository is open source and hosted on BitBucket: https://bitbucket.org/gyrokinetics/t3d.
